# CHATAPP PART 3
This is Part 3 of my ChatApp
This Chat App is a Java-based messaging system. It features user registration, secure login, and advanced message management including sending, storing, and discarding messages. Part 3 introduces constant data storage through JSON and a comprehensive management menu.
## Features
User Authentication: Secure registration and login verification.

Message Processing: Generate unique IDs and hashes for all messages.

Persistent Storage: Uses org.json to save and load messages from a local file.

Advanced Analytics: Search messages by ID/Recipient, calculate the longest message, and generate full reports.
